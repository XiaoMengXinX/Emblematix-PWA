# Emblematix PWA

A web-based PWA version of [Emblematix](https://github.com/lz233/Emblematix), built with Next.js.

A tool to embed EXIF watermarks into images.

## Features

- Embed EXIF watermarks into images
- Support for normal and compact watermark layouts
- Customizable font and watermark styles
- PWA support for offline usage
- Support for JPEG, PNG, WebP formats
